## 🎮 什么是这个平台？

这是一个**五子棋局域网对战系统**，包含两个部分：
- **游戏服务器**：负责管理玩家、房间、游戏逻辑
- **游戏客户端**：玩家使用的界面，可以在浏览器中打开

**简单来说**：就像玩网络游戏一样，你需要先启动服务器，然后打开浏览器就能和朋友下五子棋了！

## 🛠️ 开始前的准备

### 第一步：检查你的电脑是否满足要求

**必需条件**：
- Windows 10 或更高版本 / macOS / Linux
- 至少 2GB 可用内存
- 有网络连接（首次安装需要下载文件）

### 第二步：安装 Node.js

Node.js 是一个运行程序的工具，是使用这个平台的基础。

**下载和安装**：

1. 访问官网：https://nodejs.org/
2. 下载 **LTS 版本**（推荐）或 Current 版本
3. 运行下载的安装程序
4. 安装过程中一直点击"下一步"即可
5. 安装完成后，重启电脑

**验证安装**：

打开命令提示符（CMD）或 PowerShell：
- Windows: 按 `Win + R`，输入 `cmd`，回车
- macOS: 按 `Cmd + Space`，输入 `Terminal`

输入以下命令：
```bash
node -v
```

如果看到类似 `v18.17.0` 的版本号，说明安装成功！

## 📦 安装步骤

### 创建一个文件夹

在电脑上创建一个新文件夹，比如：
- Windows: `C:\五子棋游戏`
- macOS: `/Users/你的用户名/五子棋游戏`

### 安装游戏服务器

1. 打开命令提示符（CMD）或 PowerShell
2. 进入你创建的文件夹：
   ```bash
   cd C:\五子棋游戏
   ```
3. 安装服务器包：
   ```bash
   npm install @whoersir/gomoku-server
   ```

**⏱️ 这可能需要几分钟时间，请耐心等待...**

安装成功后，你会看到类似这样的提示：
```
added 1 package in 30s
```

### 安装游戏客户端

继续在命令提示符中输入：
```bash
npm install @whoersir/gomoku-client
```

**⏱️ 这也可能需要几分钟时间，请耐心等待...**

## 🚀 启动服务

### 启动游戏服务器

1. 在命令提示符中，进入安装好的服务器目录：
   ```bash
   cd C:\五子棋游戏\node_modules\@whoersir\gomoku-server
   ```

2. 创建环境配置文件：
   
   创建一个名为 `.env` 的文件，内容如下：
   ```
   PORT=3000
   ADMIN_PASSWORD=admin123
   NODE_ENV=development
   ```

   **如何创建 .env 文件**：
   - 右键点击文件夹空白处 → 新建 → 文本文档
   - 重命名为 `.env`（注意：前面有个点）
   - 用记事本打开，粘贴上面的内容
   - 保存并关闭

3. 启动服务器：
   ```bash
   npm start
   ```

   **成功提示**：
   ```
   🚀 五子棋服务器启动成功！
   📡 服务器地址: http://localhost:3000
   🎮 Socket.IO 端口: 3000
   ```

   **⚠️ 重要**：不要关闭这个命令提示符窗口！服务器正在运行中。

### 启动游戏客户端

**打开新的命令提示符窗口**（之前的不要关闭）：

1. 进入客户端目录：
   ```bash
   cd C:\五子棋游戏\node_modules\@whoersir\gomoku-client
   ```

2. 启动客户端：
   ```bash
   npm run dev
   ```

3. 等待启动完成后，会显示一个本地地址，例如：
   ```
   Local: http://localhost:5173/
   ```

## 🎮 开始游戏

### 第一个玩家连接

1. 打开浏览器（推荐 Chrome、Edge 或 Firefox）
2. 访问 `http://localhost:5173`
3. 点击"连接服务器"
4. 输入你的玩家名称，比如"小明"
5. 点击"加入"

### 创建房间

1. 在房间列表中，点击"创建房间"
2. 输入房间名称，比如"高手对战房"
3. 点击"创建"
4. 你将成为房间的第一位玩家（执黑棋）

### 第二个玩家加入

**在你的手机或另一台电脑上**：

1. 确保连接到同一个局域网（同一个WiFi）
2. 打开浏览器
3. 输入服务器的 IP 地址：
   
   **查看服务器IP地址**（在运行服务器的电脑上）：
   - 打开命令提示符
   - 输入 `ipconfig`（Windows）或 `ifconfig`（macOS）
   - 找到"IPv4 地址"，例如：`192.168.1.100`
   
   访问：`http://192.168.1.100:5173`
4. 连接服务器，输入玩家名称，比如"小红"
5. 在房间列表中选择"高手对战房"，点击"加入"
6. 你将成为第二位玩家（执白棋）

### 开始对弈

- 黑棋先手（第一位玩家）
- 轮流落子
- 五子连成一线即获胜

## ❓ 常见问题

### 问题1：启动时提示"命令未找到"

**原因**：没有安装 Node.js

**解决**：按照"开始前的准备"部分安装 Node.js

### 问题2：无法连接到服务器

**可能原因**：
- 服务器没有启动
- 端口被占用

**解决方法**：
1. 确保服务器的命令窗口没有关闭
2. 如果端口被占用，修改 `.env` 文件中的 `PORT` 为其他值（比如 3001）

### 问题3：局域网内其他设备无法访问

**原因**：防火墙阻止了连接

**解决方法**：
1. Windows：打开"Windows Defender 防火墙" → 允许应用通过防火墙
2. 找到 Node.js，勾选"专用"和"公用"

### 问题4：npm install 很慢或失败

**解决方法**：
1. 使用国内镜像源：
   ```bash
   npm config set registry https://registry.npmmirror.com
   ```
2. 重新安装：
   ```bash
   npm install @whoersir/gomoku-server
   npm install @whoersir/gomoku-client
   ```

### 问题5：浏览器提示"拒绝连接"

**检查清单**：
- [ ] 服务器是否已启动？
- [ ] 地址是否正确（`http://localhost:5173`）？
- [ ] 是否使用了 https？（应该是 http）
- [ ] 端口号是否正确？

## ⚙️ 高级配置

### 修改服务器端口

1. 编辑 `.env` 文件
2. 修改 `PORT=3000` 为你想要的端口
3. 重启服务器

### 设置管理员密码

编辑 `.env` 文件：
```
ADMIN_PASSWORD=你的强密码
```

### 跨设备连接

要让局域网内的其他设备访问：

1. 查找服务器 IP 地址
2. 其他设备访问：`http://服务器IP:5173`

**示例**：
- 服务器 IP: `192.168.1.100`
- 其他设备访问: `http://192.168.1.100:5173`

## 💡 使用技巧

### 1. 固定服务器窗口

把服务器的命令提示符窗口固定在任务栏，防止误关闭。

### 2. 创建桌面快捷方式

为客户端访问地址创建浏览器书签或快捷方式。

### 3. 多人游戏

- 每个房间支持 2 名玩家
- 其他玩家可以观战
- 支持多个房间同时运行

### 4. 保存和恢复

游戏会自动保存对局记录，可以查看历史对战。
